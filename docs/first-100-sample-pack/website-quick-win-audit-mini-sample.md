# Website Quick-Win Audit Mini - Synthetic Sample

URL reviewed: https://example.com
Goal: more quote requests from a service page

Findings:
- Mobile first screen has a clear headline but the quote button appears below a large spacer.
- Contact form asks for six fields before explaining response time.
- Alt text is missing on the main proof image.

Quick wins:
1. Move the quote button directly below the first value sentence.
2. Add "reply within one business day" above the form.
3. Add descriptive alt text to the proof image.

Acceptance:
Buyer receives visible evidence, priority, impact, and concrete next action list.
